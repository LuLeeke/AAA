생성형 AI

영상 AI 데이터/코딩 AI 음성 AI 생성형 AI 이미지 생성 AI AI 판별기 To Be Continued

아래의 AI는 텍스트, 아이디어, 이미지, 음악 등의 다양한 콘텐츠를 생성하는 만능 AI입니다.

1. Chat GPT

Chat GPT 는 OpenAI가 개발한 대화형 인공지능으로, 자연스러운 대화를 통해 정보를 제공하거나 다양한 작업을 도와주는 언어 생성 AI입니다. 텍스트 기반 작업에 매우 강력하며, 학습·코딩·글쓰기·번역 등 여러 방면에서 활용됩니다.

주요 기능

  • 텍스트 생성
    에세이, 기사, 스크립트, 광고 문구 등 다양한 글을 자연스럽게 생성합니다.
  • 질문 응답
    “경제학이 뭐야?”, “이 문제 어떻게 풀어?” 같은 질문에 대해 자세히 설명합니다.
  • 요약 및 정리
    긴 문서나 글을 핵심 내용으로 요약하거나, 글을 정돈해줍니다.
  • 다국어 번역
    여러 언어 간의 번역이 가능하며, 문맥도 잘 파악합니다.
  • 코딩 지원
    파이썬, HTML, JavaScript 등 다양한 언어의 코드 생성과 설명이 가능합니다.
  • 창작 도우미
    아이디어 브레인스토밍, 시나리오 작성, 블로그 글쓰기 등 창작 활동 지원도 탁월합니다.

2. Notion AI

Notion AI 는 메모, 문서, 프로젝트 관리에 활용되는 올인원 툴인 Notion에 통합된 인공지능 기능입니다. 글을 자동으로 생성하고 정리하며, 생산성을 극대화해주는 도우미 역할을 합니다.

주요 기능

  • 글쓰기 도우미
    블로그 글, 메모, 이메일, 회의록 등을 자동 생성합니다.
  • 문서 정리 및 요약
    긴 글이나 회의록을 요약하거나, 구조를 깔끔하게 정돈해줍니다.
  • 아이디어 브레인스토밍
    주제에 대한 다양한 아이디어를 제시해 창작 활동을 도와줍니다.
  • 할 일 목록/계획 생성
    자연어로 입력하면 할 일 목록이나 일정 계획을 자동 생성합니다.
  • 반복 작업 자동화
    동일한 형식의 문서를 빠르게 복사하고 내용만 바꿔서 생성할 수 있습니다.

3. MindMeister + AI

MindMeister + AI 는 생각을 시각적으로 정리할 수 있는 마인드맵 도구에 AI를 결합한 형태로, 아이디어 발산과 구조화 작업을 훨씬 더 빠르고 효율적으로 도와줍니다.

주요 기능

  • 자동 아이디어 생성
    중심 키워드만 입력하면 관련된 아이디어를 AI가 자동 제안해줍니다.
  • 마인드맵 구조화
    AI가 주제별로 가지를 정리해주어 구조적인 생각 흐름을 만들 수 있습니다.
  • 노트 자동 작성
    마인드맵 내용을 기반으로 요약된 노트를 자동 생성할 수 있습니다.
  • 실시간 협업 지원
    여러 사용자가 동시에 작업 가능하며, AI가 중간에 아이디어를 추가해줍니다.
  • 다양한 언어 지원
    영어, 한국어 등 다양한 언어로 마인드맵 생성이 가능합니다.

4. PromptHero

PromptHero 는 DALL·E, Midjourney, Stable Diffusion 등 다양한 이미지 생성 AI에 최적화된 프롬프트(Prompt) 예시를 제공하는 웹 플랫폼입니다. 초보자도 빠르게 원하는 스타일의 이미지를 만들 수 있게 도와줍니다.

주요 기능

  • 프롬프트 검색 기능
    원하는 스타일(예: "일러스트", "3D 캐릭터")이나 키워드로 프롬프트를 검색할 수 있습니다.
  • 모델별 프롬프트 제공
    DALL·E, Midjourney, Stable Diffusion 등 각 이미지 생성 모델에 맞는 프롬프트 예시를 제공합니다.
  • 인기 프롬프트 추천
    다른 사용자가 많이 사용한 프롬프트를 추천받을 수 있어 참고하기 쉽습니다.
  • 프롬프트 + 결과 이미지 미리보기
    프롬프트를 입력하면 어떤 이미지가 나올지 시각적으로 확인 가능합니다.
  • 사용자 프롬프트 저장
    자신만의 프롬프트를 저장하고 다시 사용할 수 있습니다.

5. Miro AI

Miro AI 는 온라인 협업 도구 Miro에 탑재된 인공지능 기능으로, 아이디어 정리, 마인드맵 생성, 회의록 작성, 요약 정리 등을 자동화해주는 스마트 도우미입니다.

주요 기능

  • 아이디어 정리 자동화
    팀 회의나 브레인스토밍 중 나온 아이디어를 AI가 자동으로 정리해줍니다.
  • 문서 요약 및 구조화
    긴 텍스트나 회의 내용을 요약하고 마인드맵 형태로 구조화해줍니다.
  • 마인드맵 자동 생성
    키워드를 입력하면 관련 아이디어를 확장해 마인드맵으로 시각화해줍니다.
  • 작업 우선순위 제안
    프로젝트의 흐름에 따라 중요한 작업을 제안하고 정리합니다.
  • 협업 기능 강화
    팀원들의 작성 내용을 분석해 유사 항목을 연결하거나 요약 제안도 가능.
